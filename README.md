# PackSmart

> Una app que genera listas de equipaje personalizadas según destino, fechas y actividades, considerando el clima y duración del viaje.

Generated by [IdeaForge](https://github.com/AgenticGroot/ideaforge) from the `spring-vue-fullstack` template.

## 🏗️ Project Structure

```
├── backend/          # Spring Boot 3 API (Java 21)
│   └── src/main/java/com/ideaforge/quickstart/
└── frontend/         # Vue 3 + TypeScript + Tailwind CSS
    └── src/
```

## 🚀 Quick Start

### Backend

```bash
cd backend
mvn spring-boot:run
```

API runs on `http://localhost:8080`

### Frontend

```bash
cd frontend
npm install
npm run dev
```

App runs on `http://localhost:5173`
